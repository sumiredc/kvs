# Key Value Store by PHP

```sh
# run local
php src/app.php <command>

# run docker container
docker compose run --rm php <command>
```

## 💻️ Features

1. `new <database>`
2. `set <key> <value> (-d <database>) (-t <string|int|float|bool>)`
3. `get <key> (-d <database>)`
4. `delete <key> (-d <database>)`
5. `list (-d <database>)`
