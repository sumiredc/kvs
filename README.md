# Key Value Store by PHP

```sh
# run local
php src/app.php <command>

# run docker container
docker compose run --rm php <command>
```

## 💻️ Features

1. `new <database>`
2. `set <key> <value>`
3. `get <key>`
4. `delete <key>`
5. `list`
6. `export <database>`
7. `import <database>`
